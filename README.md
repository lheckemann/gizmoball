# Extra features

- Multiple balls.
- Multiple collisions per tick.
- Simultaneous same-ball collisions.
- Dynamic tick spans, and a timer that adjust to them.
- Ball sinks.
- Ball spawners.
- Spinning flippers.
- Build mode cursors.
- Ball velocity vectors.
